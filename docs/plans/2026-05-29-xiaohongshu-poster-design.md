# Pi HUD Xiaohongshu Poster Design

## Goal

Create one Xiaohongshu-style poster for `Pi Agent HUD for pi-coding-agent` that can be posted directly as a cover image and also kept in an editable source format inside the repository.

## Audience

- Developers using `pi-coding-agent`
- Terminal-heavy users who care about workflow visibility and efficiency
- Users who are likely to click on "tool recommendation" or "efficiency setup" content on Xiaohongshu

## Message Priority

1. This is a terminal efficiency tool for `pi-coding-agent`
2. It turns the bottom status bar into a visible dashboard
3. It is easy to install and highly customizable

## Chosen Direction

`A | 爆款教程感`

Use a high-contrast Xiaohongshu cover layout rather than a product landing page banner. The poster should feel like a piece of shareable content first and a product visual second.

## Copy

- Brand line: `Pi Agent HUD for pi-coding-agent`
- Main headline: `把终端状态栏`
- Highlight headline: `做成效率仪表盘`
- Supporting phrases: `终端党效率神器` `信息密度直接拉满`
- Feature tags: `安装简单` `插件扩展` `自定义布局`

## Visual System

- Format: portrait Xiaohongshu cover
- Primary color: Xiaohongshu-style red
- Background: warm off-white instead of black
- Content proof: a terminal HUD card in the middle using the real project messaging style
- Decoration: sticker-like labels, rounded cards, bold text blocks, soft shadows

## Deliverables

- Editable source: `docs/posters/pi-agent-hud-xiaohongshu-poster.svg`
- Shareable export: `docs/posters/pi-agent-hud-xiaohongshu-poster.png`
